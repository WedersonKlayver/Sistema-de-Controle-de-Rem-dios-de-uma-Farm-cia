# 💊 Sistema de Gerenciamento de Remédios

Projeto desenvolvido em linguagem C para gerenciar o estoque de uma farmácia.

## 🚀 Funcionalidades

- Cadastro de remédios
- Listagem de remédios
- Busca por nome
- Busca por código
- Venda de remédios
- Remoção de remédios
- Uso de memória dinâmica (malloc e realloc)

## 💻 Tecnologias

- Linguagem C

## 📚 Conceitos utilizados

- Structs
- Ponteiros
- Alocação dinâmica de memória
- Manipulação de strings
- Estruturas de repetição e decisão
